# Simple SPA with User CRUD and KakaoMap
- Vue3, Vite, User CRUD
- 카카오맵 API를 통해 사용자 주소와 위치 업데이트
- Vue Router를 사용해 라우팅 관리 및 SPA 구현
- Vuex를 사용해 상태관리
- Swiper, Bootstrap, Sass
